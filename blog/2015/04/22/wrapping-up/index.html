Wrapping Up

Apr 22nd, 2015 1:55 pm

So, I’ve reached a place in my school work where I’m quite ahead. I have completed all my assignments for Javascript so far, though more assignments will be written soon. Now I’m learning a bit of jQuery, which has been a nice change up. I am also freshening up on my HTML and CSS skills so I can finish up my personal website. It’s been quite a few months since I’ve touched it, and I’m pretty rusty! But I’ve created a new mood board and wireframe, so at least I’ve got a new idea I’m working with.

I’m glad I have this time to get reacquainted with these languages - before I start interviewing for an internship at a local tech company! We will be having a personalized “job fair” in a couple weeks, which will allow us to meet representatives from different companies for which we will get to interview. I’m super excited for this next chapter of working for a real consulting company. This will be one step closer to doing what I truly aim for - graphic design! I’m so eager to create new things and learn how to do it well from professionals.

Although I’m thrilled for this new experience, I’m also realizing how incredibly short of time we have to prepare portfolios! Better get back to work!
